## 透過 vite 建立 React 專案

```
npm create vite@latest
```

```
npm install
npm run dev
```

<br>

## 安裝 Bootstrap、axios

```
npm i bootstrap@5.3.3 axios
```

<br>

### 在進入點引入 Bootstrap

```
import 'bootstrap/dist/css/bootstrap.min.css'; // 引入 CSS
```
